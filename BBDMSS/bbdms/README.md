# Blood Bank & Donor Management System (BBDMSS)

A web-based platform for managing blood bank operations, donor registrations, blood requests, and more.

## Features
- Donor registration and profile management
- Blood group management
- Blood request and approval workflow
- Admin dashboard for managing donors, requests, and blood groups
- Contact and support pages

## Technologies Used
- PHP
- HTML, CSS, JavaScript
- Bootstrap (for UI)
- MySQL (backend, not included in this repo)

## Folder Structure
- `/admin` — Admin dashboard and management tools
- `/includes` — Shared PHP includes (header, footer, config)
- `/css`, `/js`, `/images`, `/webfonts` — Static assets
- Main PHP files for user-facing features in the root directory

## Setup Instructions
1. Clone this repository.
2. Import the database (not included here) into your MySQL server.
3. Update `includes/config.php` with your database credentials.
4. Run the project on a local or web server with PHP support.
